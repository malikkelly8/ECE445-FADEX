## Entry 4: Post-Break Hardware Bring-Up Plan

**Date:** 3/22/26  

**Objective:** Prepare a hardware testing plan for when the team returned from spring break.

**Work Completed:**  
The team outlined the hardware bring-up tasks that would need to happen after break. These included verifying ESP32 programming, checking power rails, testing pump drivers, confirming pump direction and flow, reading load-cell values through HX711 amplifiers, generating PWM for the UV LED, and reading ADC voltage from the photodiode circuit. The goal was to test each subsystem independently before combining them into the full Home Base system.

**Results / Decisions:**  
The team decided that the post-break hardware plan should be modular and evidence-based. Each test should produce a measurable result, such as a serial monitor reading, oscilloscope waveform, measured voltage, pump output volume, or load-cell calibration value. This would make debugging easier and provide material for verification documentation.

**Next Step:**  
Return from break prepared to run subsystem tests, document results, and update the Kanban board with completed hardware milestones.
